Kendala Mengakses Dashboard?

Jika Anda mengalami kendala saat mengakses dashboard, silakan coba salah satu solusi berikut.

Gunakan Browser Lain

Coba akses dashboard menggunakan browser alternatif seperti Mozilla Firefox, Microsoft Edge, atau Safari untuk hasil terbaik.

Firefox
Edge
Safari

Gunakan Mode Samaran

Buka browser Anda dalam mode Incognito / Private untuk menghindari cache dan ekstensi yang mungkin mengganggu akses dashboard.

Chrome / Edge Ctrl + Shift + N
Firefox Ctrl + Shift + P

Bersihkan Cache Browser

Hapus cache dan cookies browser Anda, lalu muat ulang halaman dashboard. Ini sering menyelesaikan masalah tampilan dan akses.

Hard Reload Ctrl + Shift + R

Dashboard dioptimalkan untuk browser yang memilki maksimal satu akun didalamnya. Jika masalah masih berlanjut, hubungi GA Department untuk bantuan lebih lanjut.
